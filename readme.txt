For PressWizards.com clients to easily add your site logo to the login page, and a powered by Press Wizards to the login page footer, and optionally a small icon in the site footer, linking to our site using your affiliate ID.

== Description ==

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= I'm not a PressWizards.com client. Can I still use this plugin? =

Yes, of course! You don't need to be a PressWizards.com client to use this plugin.

= I'm not a PressWizards.com client. How do I become one? =

You'll head over to our website: https://www.PressWizards.com/ and request to become a client. Then we'll say welcome aboard!
